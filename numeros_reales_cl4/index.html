Operaciones con números reales:

En las operaciones con números reales, se realizan las mismas operaciones básicas que con otros tipos de números, como la suma, resta, multiplicación y división. Aquí tienes una explicación de cada una de estas operaciones:

Suma: Para sumar dos números reales, se suman sus valores numéricos. Si los números tienen el mismo signo, se suman sus valores absolutos y se conserva el signo. Si los números tienen signos diferentes, se restan sus valores absolutos y se asigna el signo del número con mayor valor absoluto.

Resta: La resta de números reales se realiza de manera similar a la suma. Se resta el valor absoluto del segundo número del valor absoluto del primer número y se asigna el signo del número con mayor valor absoluto.

Multiplicación: Para multiplicar dos números reales, se multiplican sus valores numéricos y se asigna el signo según las reglas de los signos. Si ambos números tienen el mismo signo, el resultado es positivo. Si los números tienen signos diferentes, el resultado es negativo.

División: La división de números reales se realiza dividiendo sus valores numéricos y aplicando las reglas de los signos. Si ambos números tienen el mismo signo, el resultado es positivo. Si los números tienen signos diferentes, el resultado es negativo.

Aquí tienes algunos ejemplos de operaciones con números reales:
- Suma: (-3) + 5 = 2
- Resta: 8 - (-2) = 10
- Multiplicación: 2.5 * (-4) = -10
- División: 12 / (-3) = -4

Recuerda que al realizar operaciones con números reales, es importante tener en cuenta las reglas de los signos y seguir el orden de las operaciones.
